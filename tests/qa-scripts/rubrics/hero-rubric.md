# Hero Rubric

Calibrated rubric for hero section / landing page evaluations. Used by the `design_crit` and `iteration_delta` evaluators.

## Dimensions

### Intent Match (does the design respond to the brief?)
- **3**: Text and button present but no visual connection to the product/brand
- **5**: Functional hero with headline + CTA, but generic — could be any SaaS
- **7**: Hero communicates the product's value proposition — typography, imagery, and color reinforce the brand message
- **9**: Hero with distinctive visual identity — memorable composition, brand-specific details, emotional resonance

### Visual Craft (is it curated?)
- **3**: Flat layout, no visual interest, text dumped on a background
- **5**: Basic composition with reasonable spacing, a background color or gradient
- **7**: Considered composition — visual rhythm, balanced whitespace, subtle effects (gradient, overlay, shadow on CTA)
- **9**: Polished craft — dynamic composition, layered depth, refined color transitions, attention to optical balance

### Design Decisions (are choices intentional and reasoned?)
- **3**: Default everything — centered text, generic button, no visual accent
- **5**: Some intentional choices (brand color on CTA, a decorative element)
- **7**: Clear visual strategy — primary/secondary CTA distinction, supporting visual element (illustration placeholder, pattern, shape), type scale with clear headline/subheadline/body
- **9**: Strategic decisions — visual weight distribution guides conversion, color psychology aligned with brand, typography creates distinct brand voice

### Hierarchy (does the visual hierarchy guide the eye?)
- **3**: Headline, subtitle, and CTA compete for attention — no clear flow
- **5**: Headline is dominant but subtitle and CTA lack clear ordering
- **7**: Clear flow: headline > subtitle > CTA > secondary elements. CTA is prominent without overwhelming
- **9**: Sophisticated hierarchy — eye follows the intended path naturally, every element reinforces the conversion goal

### Consistency (is it cohesive?)
- **3**: Mixed styles — button doesn't match heading style, spacing varies randomly
- **5**: Consistent within the hero but feels disconnected from a broader system
- **7**: All elements share a coherent design language — spacing grid, color roles, type scale
- **9**: Production-ready — every element feels part of a designed system, could extend to a full landing page seamlessly
